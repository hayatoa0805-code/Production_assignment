推し活管理アプリ
最終更新(26/02/09)

概要
推し活に関する情報（推し・イベント・グッズ・支出など）を
一元管理することを目的としたWebアプリケーションです。

制作のきっかけ
既存のアプリ・サイトでは、
情報収集・イベント日時の管理・収支管理などを
複数のサービスに分けて管理する必要があり、不便に感じていました。
そこで、自分が使いやすいような
押し活に関する情報を一つにまとめて管理できるサービスを作りたいと考え、
この推し活管理アプリを制作しました。

主な機能
1.ユーザー認証（会員登録・ログイン・ログアウト）
2.推し情報の登録・編集・削除
3.イベント登録・編集・削除
4.グッズ登録・編集・削除
5.収支登録・編集・削除

使用技術
・Pyhon
・Django
・HTML / CSS

今後実装予定のもの（優先順）

・スマホ対応
・日記投稿
・収支の詳細分析機能 / 登録グッズの購入、未購入の紐付け
・アカウントの設定変更
・デザインの調整・改善
