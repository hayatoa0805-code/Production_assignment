推し活管理アプリ

概要
推し活に関する情報（推し・イベント・グッズ・支出など）を
一元管理することを目的としたWebアプリケーションです。

制作のきっかけ
既存のアプリ・サイトでは、
情報収集・イベント日時の管理・収支管理などを
複数のサービスに分けて管理する必要があり、不便に感じていました。
そこで、自分が使いやすいような
押し活に関する情報を一つにまとめて管理できるサービスを作りたいと考え、
この推し活管理アプリを制作しました。

主な機能（2026/1/28日 時点）
1.ユーザー認証（新規登録・ログイン・ログアウト）
2.推し情報の登録
3.イベント登録
4.グッズ登録
5.収支登録

使用技術
・Pyhon
・Django
・HTML / CSS
